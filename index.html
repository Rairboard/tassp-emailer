Installation
Install in 4 Steps
1
Download the extension
Click the Download button above. You'll get a .zip file — save it somewhere you'll remember.
2
Unzip the file
Right-click the downloaded tassp-emailer.zip and select Extract All (Windows) or double-click (Mac). You'll get a folder called tassp-v6.
3
Enable Developer Mode in Chrome
Open Chrome and go to chrome://extensions. Toggle Developer mode on using the switch in the top-right corner.
4
Load the extension
Click Load unpacked, then select the tassp-v6 folder. The TASSP Emailer icon will appear in your Chrome toolbar.
